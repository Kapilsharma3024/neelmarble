# NeelMarble — Premium 3D Marble Showroom

Ultra-luxury marble showroom website built with Next.js, React Three Fiber, GSAP, and Framer Motion.

## Getting Started

```bash
cd E:\3d-website
npm install
npm run dev
```

Open http://localhost:3000

## Admin

- URL: `/admin`
- Password: `admin123` (override with `ADMIN_PASSWORD` env var)

## Production

```bash
npm run build
npm start
```
